Quick Start:
1. npm install
2. npm start (starts both servers)
3. Open http://localhost:3002 (login page)
4. Enter HeyGen credentials
5. Open http://localhost:3000 (control panel)
6. A browser window opens - interact with it directly!

System Architecture:
- Port 3002: Auth server (custom login UI)
- Port 3000: Control panel + WebSocket server
- Playwright Browser: Live headed browser (authenticated)

Flow:
1. User logs in at custom UI (port 3002)
2. Playwright authenticates with HeyGen
3. Session cookies saved to heygen-cookies.json
4. Open control panel at port 3000
5. Playwright opens a LIVE browser window
6. User interacts directly with the browser
7. All requests automatically authenticated!
